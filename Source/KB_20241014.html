첨부파일 다운로드
열람 안내 * 특정 앱, 브라우저는 상단 첨부파일 다운로드가 지원되지 않을 수 있습니다.

* Chrome, FireFox 브라우저 등을 이용하여 보안메일 html 파일을 다운로드 후, 다운로드된 보안메일 html 파일을 VestMail 전용뷰어 로 열람하여 주십시요.

> VestMail 전용뷰어를 통한 열람시 아래와 같은 과정으로 열람이 가능합니다.

  1) VestMail 설치.
  2) 메일 열람.
  3) 보안메일 html 파일 다운로드.
  4) 다운로드된 보안메일 html 파일 선택.
  5) 열람 가능앱 리스트에서 "VestMail" 선택.
  6) 비밀번호 입력 후 첨부파일 선택하여 파일 저장(열람).

* 보안메일을 이미 다운받은 경우, 전용뷰어(VestMail)에서 다운로드 된 파일을 확인 할 수 있습니다.

  6) 비밀번호 입력 후 첨부파일 선택하여 파일 저장(열람).
